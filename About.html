About

Fractal Design is a gallery of images created using various chaotic algorithms. The Julia sets and strange attractors are true fractals. The cellular automata on this site are mostly generated using the Belousov-Zhabotinsky algorithm or some modified form of it.

All the images displayed here were generated using software I wrote myself. I am a scientist by training and originally wrote this software to motivate me to learn to program well over a decade ago so I could make a career change. I continue to work on these programs as a hobby on side. I still find chaos and fractals a fascinating subject in itself and the products of the software an additional benefit.

It has been a while since I originally started this site in 2004 and this is the first major update since then, however I should be adding more content gradually from now on.
